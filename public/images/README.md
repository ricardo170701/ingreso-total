# Imágenes del Sistema

## Logo de la Gobernación del Meta

Coloca el logo de la Gobernación del Meta (sin fondo) en este directorio con el nombre:

**`logo-gobernacion-meta.png`**

### Especificaciones recomendadas:

-   Formato: PNG con transparencia (sin fondo)
-   Tamaño recomendado: 300-400px de ancho
-   Proporción: Mantener la relación de aspecto original del logo

El logo se mostrará automáticamente en la pantalla de login del sistema.
